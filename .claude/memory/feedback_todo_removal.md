---
name: Remove completed TODO items
description: When completing a task picked from TODO.md, delete the item from the file entirely rather than marking it done
type: feedback
---
When a task comes from TODO.md, remove the completed item from the file entirely once the work is done. Do not mark it with `[x]` or add any "done" annotation — just delete the line.

**Why:** User prefers a clean TODO list with no residue of completed work.

**How to apply:** At the end of any session where a TODO.md item was implemented, delete that line from the file before wrapping up.
